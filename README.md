GyMik - Android aplikace
=================

Android aplikace pro studenty Gymnazia v Plzni, Mikulasske nam. 23. Umoznuje mit vzdy po ruce suplovani, rozvrh a mapu skoly. Zdrojovy kod je urcen pro studijni ucely.
